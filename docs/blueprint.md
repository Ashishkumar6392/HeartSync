# **App Name**: HeartSync

## Core Features:

- Profile Creation: Create appealing user profiles with guidance on effective self-presentation.
- Match Recommendation: Recommend compatible matches based on profile information and interaction history.
- Conversation Starters: Generate engaging conversation starters tailored for video and voice chats. An LLM tool reasons from the match's profile to generate each conversation starter.
- Personality Analysis: Analyze personality traits from user's voice or text input. An LLM analyzes recorded speech and textual content to estimate the user's personality.
- Safe Dating Guidance: Provide real-time guidance on safe online dating practices and ethical interactions. LLM is the tool used to offer context-specific advice and information.
- Risk Detection: Detect toxic or risky behavior patterns in calls or messages.
- Profile Quality Improvement: Provide personalized tips on improving video/audio profile quality, including lighting, sound, confidence, and overall presentation. This uses generative AI to analyze profiles and provides advice.

## Style Guidelines:

- Primary color: Vibrant coral (#FF7F50) to convey warmth and connection.
- Background color: Soft peach (#F2D9C7), a desaturated tint of coral to ensure good contrast while retaining the feel of the primary.
- Accent color: Muted salmon (#FA8072), an analogous color to the primary to complement key interactive elements.
- Body and headline font: 'Inter', a modern sans-serif known for its legibility and neutral appearance, well-suited for on-screen reading.
- Use modern, minimalist icons to represent different features and actions, focusing on clarity and ease of understanding.
- Clean and intuitive layout with a focus on visual hierarchy to guide users through the profile creation and matching processes.
- Subtle, smooth transitions and animations to enhance user engagement and provide feedback on interactions.
